## Hotkeys + Commend lines + Git

### Google chrome Hotkeys 

Show Full screen: 
```
shift+command+F
```

Jump to next tap:
```
option+commond+right/left arrow
```

Close current tap:
```
commond+w
```

Reopen closed tap:
```
commond+shift+t
```

Choose most right/left words:
```
shift+commind+right/left arrow
```

Choose whole words:
```
option+right/left arrow
```
